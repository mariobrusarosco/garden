##### Syntax

```tsx
import { FlatList, Text } from 'react-native';

const fruits = [{ id: '1', name: 'apple' }, { id: '2', name: 'banana' }, { id: '3', name: 'cherry' }, { id: '4', name: 'date' }, { id: '5', name: 'elderberry' }];


const FruitsList = ({ data, renderItem }: { data: any[], renderItem: (item: any) => React.ReactNode }) => {
  return <FlatList data={data} renderItem={({ item }) => <Text>{item.name}</Text>} />;
};

export default FruitsList;
```

##### Key Prop

Key prop is a required prop for FlatList. It's used to identify the item in the list. 
If we don't provide a key prop, React Native will drop all the items and re-render them, which is not efficient.

1. With a string for the key prop

```tsx
const fruits = [{ id: '1', name: 'apple' }, { id: '2', name: 'banana' }, { id: '3', name: 'cherry' }, { id: '4', name: 'date' }, { id: '5', name: 'elderberry' }];
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.id}</Text>} />
```

2. With keyExtractor where we provide the key for each item
```tsx
const fruits = [{ name: 'apple' }, {  name: 'banana' }, { name: 'cherry' }, { name: 'date' }, { name: 'elderberry' }];
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.name}</Text>} keyExtractor={(item) => item.name} />
```

##### Axis


We can add margin for a axis:

We can define the direction of the items with a prop called `horizontal` or `vertical`

```tsx
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.name}</Text>} horizontal />
```

```tsx
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.name}</Text>} vertical />
```

##### Scroll Indicators

We can add scroll indicators with a prop called `showsHorizontalScrollIndicator` or `showsVerticalScrollIndicator`

```tsx
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.name}</Text>} showsHorizontalScrollIndicator />
```

```tsx
<FruitsList data={fruits} renderItem={({ item }) => <Text>{item.name}</Text>} showsVerticalScrollIndicator />
```
