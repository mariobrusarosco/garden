# React router

## Render a specific page for the Root path ("/")

### Root path

1. We set <RootLayout> as the Element. Its purpose is to work as a _Layout UI_, that will apply some stylying
   and, render child Components.
2. `{ index: true, element: <Intro /> }` does the trick. It sees if the URL is just '/'. If it is, it renders:
   `<RootLayout>
      <Intro />
 </RootLayout>` and it stops!.

If not, it renders the continuation of the URL: 1) "/table", 2)"/table/123" etc:

1.  `<RootLayout>

      <AuthenticatedLayout>
    <TableLayout>
      <NoSelectedTable />
    </TableLayout>
  </AuthenticatedLayout>
</RootLayout>`

2.  `<RootLayout>
      <AuthenticatedLayout>
    <TableLayout>
      <TableScreen />
    </TableLayout>
  </AuthenticatedLayout>
</RootLayout>`

```tsx
export const AppRouter = () => {
  const router = createBrowserRouter([
    {
      path: "/",
      element: <RootLayout />,
      errorElement: <ErrorScreen />,
      children: [
        { index: true, element: <Intro /> },
        {
          element: <AuthenticatedLayout />,
          children: [
            {
              path: "tables",
              element: <TableLayout />,
              children: [
                { index: true, element: <NoSelectedTable /> },
                { path: ":tableId", element: <TableScreen /> },
                {
                  path: ":tableId/channels/:channelId",
                  element: <ChannelScreen />,
                },
                {
                  path: ":tableId/game",
                  element: <GameScreen />,
                },
              ],
            },
          ]
        }
      ]
    }
  ])
```
