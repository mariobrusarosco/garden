# Building Blocks

## Box

It starts as a <div /> with no styling. We can start customizing it by:

- changing the html tag

```tsx
<Box component="section">This Box renders as an HTML section element.</Box>
```

- adding custom css

```tsx
<Box sx={{ p: 2, border: "1px dashed grey" }}>
  This Box renders as an HTML section element.
</Box>
```

## Container

By default it gives us a centralized <div/> with:

```css
width: 100%;
margin-left: auto;
box-sizing: border-box;
margin-right: auto;
display: block;
padding-left: 16px;
padding-right: 16px;
```

_The padding values are the ones fromt `App's theme`_
